# 隐私模式使用说明

## 功能介绍

隐私模式是一个快速保护笔记内容的功能，可以一键将编辑区域模糊化，防止他人偷看您的笔记内容。

## 如何使用

### 开启隐私模式

1. 在工具栏最右侧找到"眼睛"图标按钮
2. 点击按钮开启隐私模式
3. 编辑区域会立即变模糊

### 关闭隐私模式

1. 再次点击"眼睛"图标按钮
2. 编辑区域恢复清晰

### 临时查看内容

在隐私模式开启时：
- 将鼠标悬停在编辑区域上
- 内容会临时变清晰
- 鼠标移开后自动恢复模糊

## 功能特点

### 1. 一键切换
- 点击按钮即可快速开启/关闭
- 按钮会高亮显示当前状态
- 紫色背景 = 隐私模式已开启

### 2. 毛玻璃效果
- 使用 CSS blur 滤镜
- 模糊度：8px
- 平滑过渡动画（0.3秒）

### 3. 悬停查看
- 鼠标悬停时自动清晰
- 无需关闭隐私模式
- 方便快速查看内容

### 4. 不影响编辑
- 隐私模式不影响文字输入
- 可以在模糊状态下继续编辑
- 悬停查看后可以精确编辑

## 技术细节

### CSS 实现

```css
/* 隐私模式激活时 */
.privacy-mode-active {
    filter: blur(8px);           /* 模糊8像素 */
    transition: filter 0.3s ease; /* 平滑过渡 */
}

/* 鼠标悬停时清晰 */
.privacy-mode-active:hover {
    filter: blur(0px);
}

/* 按钮激活状态 */
#privacyBtn.active {
    background: var(--neon-purple);
    color: white;
}
```

### JavaScript 实现

```javascript
let privacyModeEnabled = false;

function togglePrivacyMode() {
    privacyModeEnabled = !privacyModeEnabled;
    const pageContent = document.getElementById('pageContent');
    const privacyBtn = document.getElementById('privacyBtn');
    
    if (privacyModeEnabled) {
        pageContent.classList.add('privacy-mode-active');
        privacyBtn.classList.add('active');
    } else {
        pageContent.classList.remove('privacy-mode-active');
        privacyBtn.classList.remove('active');
    }
}
```

## 注意事项

### 1. 不是真正的加密
- 隐私模式只是视觉上的模糊
- 不会加密或隐藏实际数据
- 技术人员仍可通过开发者工具查看

### 2. 不影响保存
- 隐私模式不影响数据保存
- 关闭浏览器后状态不会保留
- 重新打开页面需要重新开启

### 3. 性能影响
- blur 滤镜可能略微影响性能
- 在低端设备上可能有轻微卡顿
- 建议在需要时才开启

### 4. 截图保护
- 隐私模式可以防止截图泄露内容
- 但无法防止相机拍摄
- 建议配合其他安全措施

## 常见问题

### Q: 隐私模式会保存吗？
A: 不会。刷新页面后需要重新开启。

### Q: 可以调整模糊程度吗？
A: 当前版本不支持，默认为 8px。未来版本会添加此功能。

### Q: 隐私模式影响性能吗？
A: 有轻微影响，但在现代浏览器中几乎察觉不到。

### Q: 可以只模糊部分内容吗？
A: 当前版本会模糊整个编辑区域。选择性模糊在计划中。

### Q: 隐私模式安全吗？
A: 只是视觉保护，不是真正的加密。适合防止偷看，不适合保护高度敏感信息。

### Q: 为什么悬停会清晰？
A: 方便您在隐私模式下快速查看和编辑内容，无需频繁切换。

### Q: 可以设置快捷键吗？
A: 当前版本不支持，但在计划中。
